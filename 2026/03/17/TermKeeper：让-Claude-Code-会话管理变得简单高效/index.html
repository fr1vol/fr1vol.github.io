TermKeeper：让 Claude Code 会话管理变得简单高效

John Doe
2026-03-17
工具
AI, Claude, Python, 工具

一个开源的 Claude Code 日志管理工具，让你的 AI 对话永不丢失，一键恢复上下文。

背景

作为一个重度使用 Claude Code 的开发者，我经常遇到一个问题：

每次重新打开 Claude Code 时，之前的对话上下文都丢失了。

我记得上周在做一个项目，和 Claude 讨论了很久的架构设计。过了几天后想继续，但完全忘记了当时的讨论进度和结论。我不得不手动翻看历史记录，重新理解之前的上下文…

这不仅浪费时间，更重要的是打断思路。

市面上的方案

我调研了一下现有的解决方案：

方案 优点 缺点
手动复制粘贴 简单 费时费力，容易遗漏
截图保存 直观 无法搜索，无法恢复上下文
笔记软件整理 结构化 需要大量手动工作
TermKeeper 自动化 + 智能压缩 需要配置 LLM API

TermKeeper 是什么？

TermKeeper (tk) 是一个用 Python 编写的 Claude Code 日志管理工具，它可以：

  1. 自动同步 Claude Code 的所有会话
  2. 智能压缩 对话内容（718:1 的压缩比！）
  3. 一键恢复 会话上下文
  4. 纯 Markdown 格式，方便搜索和管理
